# OptirVision DTF — website

Marketing site for **OptirVision DTF** — edge defect inspection for DTF printers.
Static HTML, no build step.

## Pages
- `index.html` — landing page
- `privacy.html` — privacy policy

## Run locally
```bash
python3 -m http.server 8000   # → http://localhost:8000
```

## Deploy
Any static host (Cloudflare Pages / Netlify / GitHub Pages). Build command: none.
Output directory: repository root.
